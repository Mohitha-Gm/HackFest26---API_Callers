# 🤖 Autonomous Career Optimization Agent

An AI-powered autonomous job tracking system that turns your job search into a data-driven, self-optimizing process. This agent analyzes your applications, detects stagnation, and automatically executes follow-ups and cold outreach — without manual intervention.

---

## 🚀 Features

### 📊 Smart Application Tracking
- Fetches job applications from Notion
- Tracks company, role, resume version, status, and application date
- Monitors follow-up and cold email activity

---

### 🧠 AI Strategy Analysis (Gemini)
- Identifies which resume versions perform best
- Detects which roles generate more interviews
- Calculates conversion rate
- Provides actionable strategy insights

---

### ⏳ 7-Day Stagnation Detection
- Automatically detects applications with no response after 7 days
- Generates personalized follow-up reminders

---

### 📨 Autonomous Cold Outreach
- Fetches HR contact details
- Generates personalized cold emails using resume context
- Sends emails directly to companies
- Updates status in Notion automatically

---

### 📈 Live Dashboard
- Conversion rate tracking
- Application distribution insights
- AI-generated strategic recommendations
- Cold outreach statistics

---

## 📌 Example Workflow

1. You apply to 20 jobs.
2. The system tracks everything in Notion.
3. After 7 days, 6 applications show no response.
4. The AI:
   - Drafts contextual follow-up emails  
   - Sends cold outreach to HR contacts  
   - Updates tracking automatically  
5. The dashboard shows:
   - Resume performance trends  
   - Conversion rate  
   - Strategic improvement suggestions  

---

## 🎯 What This Achieves

Instead of guessing why you’re not getting interviews, the system:

- Identifies weak resume-role alignment  
- Detects stagnation early  
- Executes outreach automatically  
- Optimizes your job search strategy over time
